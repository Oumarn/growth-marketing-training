SaaS Case Pack — Données Synthétiques pour Buyer Persona Workshop

Fichiers :
- product_funnel.csv
- analytics_traffic.csv
- crm_accounts.csv
- support_tickets.csv
- interview_guide.md
- inapp_survey.md

Rappels :
- Activation rate = first_booking / signups
- Stickiness = DAU / MAU
- LTV (approx.) = (ARPU × marge %) / churn mensuel
- LTV:CAC (cible) ≈ 3:1

Consignes atelier :
1) Recherche (15’) : repérez 3 patterns persona/funnel, 2 insights trafic, 2 tags support prioritaires.
2) Création (20’) : créez 2–3 personas + messaging (Hook • Value • Proof • CTA), choisissez l’événement d’activation & cible TTFV.
3) Validation (10’) : 2 hypothèses/test par persona + KPIs (activation rate, TTFV, D7/D30, paid).
